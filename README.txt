A modern, interactive Number Guessing Game built using Java Swing.

This application features a clean UI, rounded components, dynamic difficulty scaling, and a full guess-history table. Designed as a polished desktop application showcasing Java GUI programming.

Dynamic Game Logic
- User chooses custom range
- Number of attempts calculated automatically using logarithmic difficulty scaling 
- Immediate feedback: Too High, Too Low, Correct  

Guess History Tracking
- Every guess is logged
- Table displays:
     Guessed number
     Result (Correct / Too High / Too Low)
- Table is cleared automatically when a new game starts

How to run:

1. Save the file.
2. Open Terminal in the saved file's folder.
3. Compile Using:
    javac NumberGuess.java
4. Run Using:
    java NumberGuess
